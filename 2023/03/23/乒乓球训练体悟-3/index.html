乒乓球训练体悟 3

Posted on 2023-03-23 Edited on 2023-05-24 In Table Tennis Views:

记录 2023.03.19

这一天晚上训练内容和平时基本差不多

  1. 正手近台攻球
  2. 正手中远台拉球
  3. 反手近台反撕
  4. 发下旋球，搓球，正手起下旋衔接正手连续拉球

但是今晚进入状态非常之快，正手连续拉球质量很高，而且手感球比较多；反手相对差一点，一开始感觉不太能够控制得住球

不论什么时候，永远要将这几个词牢记在心里，在训练和实战中贯彻落实：步伐，重心，腰胯，动作小巧紧凑，发力集中

手感突然很好？

在训练的前一天，我刚好在宿舍进行正手徒手训练，主要体会如何协调的使用手和腰胯，体会挥拍与腰胯转动的协调性

受此启发，现在每天晚上都可以进行5到10分钟正手徒手训练，观察这是否对手感的建立和保持有明显的效果

步伐训练？

虽然手上感觉明显变好了，步伐也相应有些提升，但是仍然感觉调动偏慢。后续可以利用绳梯进行灵敏度训练

乒乓球步伐训练

反手

  1. 调整板型，握拍放松（正手握拍可以抓紧一点，这样发力比较顺畅，但是反手不行，抓拍太紧不利于发力）
  2. 反手首先要控制重心
  3. 动作小巧，发力集中
  4. 发力有点从右往左的感觉，不要直上直下
